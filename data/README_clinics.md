# База ветклиник (vet_clinics.json)

Файл `vet_clinics.json` — список ветеринарных клиник для отображения на карте. Его можно редактировать вручную и дополнять новыми клиниками.

## Формат

Массив объектов. Каждая клиника — объект с полями:

| Поле     | Тип    | Описание |
|----------|--------|----------|
| id       | строка | Уникальный идентификатор (например "1", "2") |
| name     | строка | Название клиники |
| city     | строка | Код города: `moscow` или `spb` (для других городов добавьте код и при необходимости донастройте фронт) |
| address  | строка | Адрес |
| lat      | число  | Широта (WGS84) |
| lon      | число  | Долгота (WGS84) |
| hours    | строка | Режим работы |
| phone    | строка | Телефон или "Н/Д" |

## Пример одной записи

```json
{
  "id": "14",
  "name": "Название клиники",
  "city": "moscow",
  "address": "ул. Примерная, 1",
  "lat": 55.75,
  "lon": 37.61,
  "hours": "Круглосуточно",
  "phone": "+7 (495) 123-45-67"
}
```

## Добавление нового города

1. Добавьте в `vet_clinics.json` клиники с полем `"city": "код_города"` (например `ekb` для Екатеринбурга).
2. В интерфейсе (выпадающий список «Выберите город») при необходимости добавьте новый пункт с `value="код_города"`.
3. В `static/js/app.js` в объект `cityCenters` добавьте центр и зум карты для нового города, например:  
   `ekb: { center: [56.84, 60.61], zoom: 10 }`.

После изменений сохраните файл и обновите страницу.
